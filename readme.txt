Step 1) open "http://localhost:8086/h2-console" //I have used h2 mem database to store the data
Step 2) run the project as "Spring Boot Project"
Step 3) POST URL is: "http://localhost:8086/api/loadTrainingCampDetail"
Step 4) GET URL for details of all Training Centers is: "http://localhost:8086/api/getTrainingCampDetail"
Step 5) GET URL for details of specific center is: "http://localhost:8086/api/getTrainingCampDetail/centerCode/enter-centerCode-here"
